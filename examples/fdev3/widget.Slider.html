fdevlib-example: slider（幻灯片效果）

1、自动无效果切换

调用代码示意：

						var mar1 = new FD.widget.Slider('mar1');

2、自动滚动效果切换，间隔5秒。鼠标移动即触发

调用代码示意：

					var slider2 = new FD.widget.Slider('slider2',{
						effect: 'scroll',
						eventType: 'mouse',
						timeDelay: 5 
					});

3、淡入淡出效果

调用代码示意：

						var slider3 = new FD.widget.Slider('slider3',{
							effect: 'fade'
						});
