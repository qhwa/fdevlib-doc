reset：重置浏览器样式

reset文件用来重置浏览器样式，统一各大浏览器的默认表现形式，解决部分兼容性问题。

            新特性：

        
和老版本相比，新版reset在以下方面进行了改进：
            
    
                
  • 1、默认的字体行高设为1.5，避免了部分字体显示不全的bug。同时无单位的行高设定也便于别的应用的行高设定。详见 Russ Weakley 的精彩讲解:《 Line Height 》。
    • 
                
  • 2、修改了老版本中浏览器默认纵向滚动条的设定方式设定，改为：html { overflow-y: scroll; }，更加友善的写法，防止因滚动条引起的闪烁。
    • 
                
  • 3、重新加上:focus{outline:0;}的设定，原因是现阶段Tab操控用户少，暂不予考虑。同时可以确保firefox下面链接在被点击状态不出现超长的边框线。
    • 
                
  • 4、根据中文站的视觉规范，直接设置a标签hover状态的颜色为#ff7300。
    • 
                
  • 5、考虑到中文站ie7用户的比例很大，直接在reset中增加了ie7的私有属性：img{ -ms-interpolation-mode:bicubic; }设置，相关说明可见《用私有属性来拯救IE7缩放图片的失真》。
    • 
            

            注意： 没有万能的银弹，不要寄希望于reset解决你的所有问题。 

        

            代码：

        
		/**
		 * reset.css
		 * 重置浏览器的常用样式
		 * @version 20100510
		 */
		 
		/* 设置页面默认的白底黑字以及默认纵向滚动条位置 */
		html{
			color: #000;
			overflow-y: scroll;
			background: #fff;
		}

		/* 清除内外边距 */
		body, h1, h2, h3, h4, h5, h6, hr, p, blockquote,  /* structural elements 结构元素 */
		dl, dt, dd, ul, ol, li, /* list elements 列表元素 */
		pre, /* text formatting elements 文本格式元素 */
		fieldset, lengend, button, input, textarea, form, /* form elements 表单元素 */
		th, td { /* table elements 表格元素 */
		    margin: 0;
		    padding: 0;
		}

		/** 设置默认字体及大小 **/
		body, button, input, select, textarea {
		    font: 12px/1.5 Tahoma, Arial, "\5b8b\4f53", sans-serif; /* 中文用 ascii 字符表示，使得在任何编码下都无问题 */
		}
		h1, h2,	h3,	h4, h5, h6 { 
			font-size:100%; 
			font-weight:normal; 
		}
		address, cite, dfn, em, var { font-style: normal; } /* 将斜体扶正 */
		code, kbd, pre, samp, tt { font-family: "Courier New", Courier, monospace; } /* 统一等宽字体 */
		small { font-size: 12px; } /* 小于 12px 的中文很难阅读，让 small 正常化 */


		/** 重置列表元素 **/
		ul, ol { list-style: none; }

		/** 重置文本格式元素 **/
		a { text-decoration: none; }
		a:hover { 
			text-decoration: underline;
			color:#ff7300; 
		}
		abbr[title], acronym[title] { /* 注：1.ie6 不支持 abbr; 2.这里用了属性选择符，ie6 下无效果 */
			border-bottom: 1px dotted;
			cursor: help;
		}
		q:before, q:after { content: ''; }
		:focus{	outline:0;} 	/* firefox下面链接在被点击状态不出现超长的边框线*/

		/** 重置表单元素 **/
		legend { color: #000; } /* for ie6 */
		fieldset, img { border: none; } /* img 搭车：让链接里的 img 无边框 */
		/* 注：optgroup 无法扶正 */
		button, input, select, textarea {  font-size: 100%; /* 使得表单元素在 ie 下能继承字体大小 */}

		/** 重置表格元素 **/
		table {
			border-collapse: collapse;
			border-spacing: 0;
		}
		td,th{
			border:1px solid #000;
		}

		/** 统一hr为1像素的横线 **/
		hr {
		    border: 0;
		    color: #000;
		    background-color: #000;
		    height: 1px
		}

		/** 解决ie7下图片缩放的失真问题 **/
		img{ -ms-interpolation-mode:bicubic; }
