grid：中文站栅格系统

grid根据中文站的特定的页面布局整理的两种栅格系统。

更新日志

  • 2010-07-26 1.0beta版本
  • 2010-07-30 1.0RC版本

开始：

如果你想引用grid，请在页面中加入以下片段片段: 
		@import url("http://style.china.alibaba.com/css/lib/fdev-v3/grid/grid-min.css");
如果你引用的是fdevlib的css合集（包含grid在内的所有全站公用css样式），请在页面中加入以下代码片段： 
		@import url("http://style.china.alibaba.com/css/lib/fdev-v3/fdev.css");

新特性：

同老版本相比，作如下调整：

  • 1、去掉#doc.auto定义全屏宽度
  • 2、改#doc.w952为w952来定义标准页宽
  • 3、改#hd为#header
  • 4、改#bd为#content
  • 5、改#ft为#footer

栅格实现：

grid-32系统（适用于市场及社区相关页面）:

1、 class名：.layout-p32a24(以32px为最小栅格单位，8px为间隔，共计24个栅格单位)；

2、 grid-32栅格系统可构建的布局大小：

class =.grid-1.grid-2.grid-3.grid-4.grid-5.grid-6 .grid-7.grid-8.grid-9.grid-10.grid-11.grid-12
W(px) =3272112152192232 272312352392432472
class =.grid-13.grid-14.grid-15.grid-16.grid-17.grid-18 .grid-19.grid-20.grid-21.grid-22.grid-23.grid-24
W(px) =512552592632672712 752792832872912952

3、使用范例:
a)、两栏式用法: 

		

			
632px(.grid-16)

			
312px(.grid-8)
b)、三栏式用法: 
		

			
312px(.grid-8)

			
312px(.grid-8)
	
			
312px(.grid-8)

class =.grid-1.grid-2.grid-3.grid-4.grid-5.grid-6 .grid-7.grid-8.grid-9.grid-10.grid-11.grid-12.grid-13.grid-14.grid-15

class =.grid-16.grid-17.grid-18.grid-19.grid-20.grid-21 .grid-22.grid-23.grid-24.grid-25.grid-26.grid-27.grid-28.grid-29.grid-30
