core：javascript核心层

整个fdevlib的javascript基于YUI2.8版本的架构。core里放的就是底层框架的文件以及一些对应的封装。分为fdev.js主文件和yui目录。

fdev.js(fdev-min.js): fdevlib的核心文件，当使用fdevlib的js框架时必须引用。

yui目录：需要用到的其他yui的Utility，比如animation，json等。

开始：

如果你想引用fdev.js，请在页面中加入以下片段片段: 
		ImportJavscript.url("http://style.china.alibaba.com/js/lib/fdev-v3/core/fdev-min.js");
注意： fdev.js(fdev-min.js)是fdevlib框架的核心文件，在使用fdevlib的时候必须引入此文件。

概要说明

1、fdev.js主要包含以下几个部分：

  • a、YUI 2.8.1 global+dom+event；
  • b、YUI 2.8.1 event-mouseenter：YUI2.8中新增的mouseenter和mouseleave事件；
  • c、YUI 2.8.1 event-delegate：YUI2.8中新增的事件代理模式；
  • d、YUI 2.8.1 Selector Utility：css选择器，自行修复了在ie8兼容模式下失效的问题；
  • e、FDEV的全局定义和$的快捷缩写；
  • f、Array的原型扩展；
  • g、String的原型扩展；
  • h、FD.common常用方法集。

2、全局命名空间替换：FDEV
var FDEV = YAHOO; 根据完全隐藏底层框架的原则，代码中不应该再暴露YAHOO的命名空间，遇到需要使用YAHOO.lang等实现未定义的命名空间时统一转换成FDEV.lang。

3、FD命名空间

		FD.namespace = function() {
			var args = Array.prototype.slice.call(arguments, 0), i;
			for (i = 0; i < args.length; ++i) {
				if (args[i].indexOf('FD') != 0) {
					args[i] = 'FD.' + args[i];
				}
			}
			return YAHOO.namespace.apply(null, args);
		}
		FD.namespace('core','util','widget');
定义fdevlib的命名空间FD。同时先注册FD.core、FD.util 和 FD.widget三个命名空间。

4、$快捷方法介绍： 为了提高开发效率。我们用$符提供了最常用的方法缩写。当使用外链js时可以使用。但是如果直接在页面上编写js时，$符号会和模版语言突出，不能使用快捷写法。

  • $Y = FYU = FDEV.util
  • $D = FYD = FDEV.util.Dom
  • $E = FYE = FDEV.util.Event
  • $$ = FYS = FDEV.util.Selector.query
  • $ = FYG = FDEV.util.Dom.get
  • $A = FTA (将队列转化为数组)

5、Array/String原型扩展：
通过Array的原型扩展实现了javascript 1.6的新特性(参见：New_in_JavaScript_1.6)和部分Array和String对象常用的方法。

6、FD.common方法
常用方法合集，编码url，复制属性等。
