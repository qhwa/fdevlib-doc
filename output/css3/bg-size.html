background-size

定义：

Name: background-size
值: <bg-size> [ , <bg-size> ]*
初始值: auto
适用于: 所有元素
继承性: no
百分比: 支持，详细看下面
媒体: visual
计算值: 按照指定值或者按照定义

指定值:

        <bg-size> = [ <length> | <percentage> | auto ]{1,2} | cover | contain

此属性用来定义背景图的尺寸。

contain:
等比例缩放背景图，达到背景图不超出背景位置区域的最大尺寸。
cover:
等比例缩放背景图，达到背景图可以把背景位置区域铺满的最小尺寸。
[ <length> | <percentage> | auto ]{1,2}:
第一个参数指定背景图的宽度，第二个参数指定高度。如果只有一个参数，则第二个参数默认是 auto.
百分比是相对于背景位置区域的大小的。
如果某个值是auto，则根据另外一个值，保持图片本身的长宽比例，计算出这个值。如果这样计算失败，则使用图片本身的尺寸，如果依然失败，把它做 100% 对待。
如果两个值都是auto,则使用图片本身的宽度和高度。如果图片没有固有的高度和宽度属性，则把它做contain 对待。
不允许有负值。

兼容性 (√)chrome (√)firefox 3.6+ (√)opera 10.6+ (√)safari 5+ (√)IE 9

说明：

firefox 3.6 中，需要通过 -moz- 开头的浏览器自定义属性来获得支持。在 firefox 4 中，开始使用标准的属性和值。

code:

        <div class="show">
            <div class="contain"></div>
            <div class="cover"></div>
            <div class="value"></div>
            <div class="percentage"></div>
            <div class="auto1"></div>
            <div class="auto2"></div>
            <div class="auto3"></div>
            <img src="/fdevlib/static/img/css3/water.png" alt="water" />
        </div>
        <style type="text/css">
            .show { display:inline-block; }
            .show div {
                float:left;
                width:100px;
                height:100px;
                padding:10px;
                border:10px dashed #666;
                background:url(/fdevlib/static/img/css3/water.png) no-repeat;
                margin:8px;
                background-position:0 0;
            }
            .show .contain {
                -moz-background-size: contain;
                background-size: contain;
            }
            .show .cover {
                -moz-background-size:cover;
                background-size:cover;
            }
            .show .value {
                -moz-background-size:80px 50px;
                background-size:80px 50px;
            }
            .show .percentage {
                -moz-background-size:50% 50%;
                background-size:50% 50%;
            }
            .show .auto1 {
                -moz-background-size:80px;
                background-size:80px;
            }
            .show .auto2{
                -moz-background-size:50% auto;
                background-size:50% auto;
            }
            .show .auto3{
                -moz-background-size:auto auto;
                background-size:auto auto;
            }
        </style>
