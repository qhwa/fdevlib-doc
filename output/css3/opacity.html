opacity

定义：

Name: opacity
值: <alphavalue> | inherit

alphavalue : 透明度； 0.0 (fully transparent) - 1.0 (fully opaque) 之间的浮点数

浏览器兼容性：

浏览器 CHROME FIREFOX OPERA SAFARI IE
兼容性 (√)chrome 5+ (√)firefox 3.6+ (√)opera 10.6+ (√)safari 5+ (×)IE6-8; IE9 unverified

说明：

由浮点数字和单位标识符组成的长度值。不可为负值。声明用来设置一个元素的透明度，opacity取值为1的元素是完全不透明的，反之，取值为0是完全透明的，看不见的。1到0之间的任何值都表示该元素的透明程度。

例子：

当前浏览器显示效果：

我是黑色不透明的。

我是黑色半透明的

我有黑色背景

我有黑色背景&我是半透明的

chrome效果截图：

code:

        <div class="show">
            <p class="black">我是黑色不透明的。</p>
            <p class="black-trans">我是黑色半透明的</p>
            <p class="black-back">我有黑色背景</p>
            <p class="back-trans">我有黑色背景&我是半透明的</p>
        </div>
        <style type="text/css">
            .show { padding:10px; width:500px; background-color:#0078AA;  }
            .show .black { color: #000; }
            .show .black-trans { color: #000; opacity:0.5; }
            .show .black-back { color: #FFF; background-color:#000; }
            .show .back-trans { color: #FFF; background-color:#000; opacity:0.5; }
        </style>
