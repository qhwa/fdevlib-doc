text-shadow

定义：

Name: text-shadow
值: none | [<shadow>, ] * <shadow>
初始值: none
适用于: 所有元素
继承性: yes
百分比: N/A
媒体: visual
计算值: 每一个 <length>都是绝对值，颜色都是计算的，其他按照规定
Name: shadow
值: inset? && [ <length>{2,4} && <color>? ]

<shadow>的定义和 box-shadow 中的 <shadow> 定义一样，但是不允许 inset 关键字。

浏览器兼容性：

浏览器 CHROME FIREFOX OPERA SAFARI IE
兼容性 (√)chrome (√)firefox 3.6+ (√)opera 10.6+ (√)safari 5+ (×)IE 9

说明：

从测试结果看来不支持<shadow>中的第四个长度值，即延伸距离
