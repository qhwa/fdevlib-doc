ui-combobox : 模拟下拉菜单

介绍

实现单选，多选类型的下拉模拟，灵活性比较大，实现简单，但是需要遵循一定的规则。必要情况下需要自定义样式

开始

//模块动态载入
jQuery(function($){
    $.use('ui-combobox',function(){
        //...
    });
});

Configs

name : String
Default : ""
对应提交时的name值
multiple : Boolean
Default : false
同时显示几个月
closable : Boolean
Default : false
是否出现关闭按钮，出现关闭按钮后无法通过点击下拉层外自动关闭下拉
editable : Boolean
Default : false
功能暂不可用
maxHeight : Int | Boolean
Default : false
指定下拉层的最大高度，默认为不限制高度。限制高度后，内容超出高度会出现滚动条
bgiframe : Boolean
Default : true
triggerText : String
Default : ""
下拉箭头的文本显示
tpl : Function
Default : 
function(data){
    var ul = $('<ul>'), i, j;
    for (i = 0, j = data.length; i < j; i++) {
        var item = data[i], li = $('<li>').addClass('ui-combobox-item').data('item', item);
        ul.append(li.html(item.text));
    }
    return ul;
}
下拉列表模板， TODO：需要对单个项增加 'ui-combobox-item' 样式，且设置 data('item') 为当前项的数据。
itemNode : Tag 的 String
Default : 'li'
列表项的节点字符串形式
itemTpl : Function
Default : 
function(item){
    return $('<li>').html(item.text);
}
列表中项的模板
resultTpl : Function
Default : 
function(item){
    return item.text;
}
结果项的模板
zIndex : Int
Default : 1000
附加到下拉列表的z-index样式

Methods

.toggle()
展开/收拢下拉列表
.resetPanelPosition()
下拉框位置重新计算
.reset( [data] )
  • data : Object 可选，指定新的数据源
重置下拉控件
.select( value[, isChanged] )
  • value : String 要选择的项的值
  • isChanged : Bolean 是否触发change事件
选择选中项
.val( [value] )
  • value : String 要设置的项的值，不传递此参数为获取值
获取/设置值
.remove( value )
  • value : String||Array 需要取消选中的项的值
Multiple时取消选中项

Events

datainit
数据准备结束时触发
listrender
下拉列表渲染结束时触发
change
下拉选中的值发生变化时触发
