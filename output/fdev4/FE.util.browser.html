web-browser : 浏览器功能插件

介绍

提供“加入收藏夹”、”设为首页“两个方法，部分浏览器不兼容。

版本

  • 施丹侠：创建
  • 倪朱国：修改----2011.11.03

开始

//手动注册模块。由于该模块使用率不高，所以没有在框架核心中注册次组件。需要使用的同学在产品线或者项目global文件中注册模块。
jQuery.add("web-browser", {
    js: ["http://style.china.alibaba.com/js/lib/fdev-v4/widget/web/browser-min.js"],
    ver: "1.0"
});
//模块动态载入
jQuery(function($){
    $.use("web-browser",function(){
        //...
    });
});

Statics

FE.util.setHome( url )
  • url : String 需要设为首页的URL
设为首页
FE.util.addBookmark( url, title )
  • url : String 需要收藏的URL
  • title : String 需要收藏的书签标题
添加到收藏
